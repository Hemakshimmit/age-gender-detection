# age-gender-detection

A deep learning-based web application that detects **age** and **gender** from images using OpenCV DNN and pre-trained Caffe models. Built with an interactive Streamlit interface.

---

## Features

- Face detection using OpenCV DNN
- Age prediction (8 age groups)
- Gender classification
- Upload image and get instant results
- Simple web interface using Streamlit

---

## Tech Stack

- Python
- OpenCV (DNN module)
- Caffe Models
- Streamlit
